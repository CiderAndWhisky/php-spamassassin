PHP SpamAssassin
=====
## Installation

### Install via Composer

Add the following lines to your `composer.json` file and then run `php composer.phar install` or `php composer.phar update`:

```json
{
    "require": {
        "templateria/spamassassin": "dev-master"
    }
}
```
